## Featured Projects

**[Maya](https://github.com/OarisaR/Maya)** — Smart Maternal Health Assistant
[![Website](https://img.shields.io/badge/Website-000000?style=flat-square&logo=googlechrome&logoColor=white)](https://maya.orebayet.workers.dev) [![Video](https://img.shields.io/badge/Video-000000?style=flat-square&logo=youtube&logoColor=white)](https://www.youtube.com/watch?v=5fSn9PKbyHI)
